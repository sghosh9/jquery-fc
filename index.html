jQuery-FusionCharts
An easy-to-use, lightweight jQuery component for the FusionCharts JavaScript Charting Library
The jQuery-FusionCharts component lets you easily include FusionCharts in your jQuery projects and add interactive charts to your jQuery applications.
Features
  • Allows to include interactivity between javascript charts.
  • Offers advanced control by giving you access to the complete FusionCharts object (that contains the chart configuration).
